
Module: Google Optimize
Author: John Burch <http://drupal.org/user/85918>


INTRODUCTION
------------

Adds Google Optimize and the Hide Page script to your website.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/google_optimize

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/google_optimize

REQUIREMENTS
------------

This module requires the following modules:

 * Google Analytics (https://www.drupal.org/project/google_analytics)

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------

 * Configure at /admin/config/system/google_optimize
