INTRODUCTION
======================
When enabled, a new field will be added to Media entity and
special filters will be added to existing Media pages.

REQUIREMENTS
======================
Drupal core 8.7
Media (core)
Media library (core)

INSTALLATION
======================
1. Enable module

CONFIGURATION
======================
1. Create taxonomy to hold directory structure.
2. Select taxonomy from the settings: /admin/config/media/media_directories
